# 登場人物：教会・聖女

## 「聖」称号者・主要人物

## 聖ソルヴェイ（現No1）

| 項目 | 内容 |
|------|------|
| 名前 | **ソルヴェイ**（語源：古北欧語 *sol*＋*vei*「太陽の力の道」） |
| 称号 | 現在の「聖」最上位（No1） |
| 年齢 | 30歳 |
| No1就任 | 4年前（26歳時）に純粋な能力で推挙 |
| 奇跡の力 | モルダを上回る。上級禍者との戦闘で英雄級の騎士・戦士と並び称されるほどの活躍 |
| 教会内の立場 | 高司祭・最高司祭級の決裁権を持つ |
| 人物像 | 冷静な観察者。力があるからこそ動かない |
| 婚姻 | 未婚 |
| 年齢（修正） | 28歳（メインストーリー時）。二十四騎討伐戦当時18歳 |
| 娘 | ソルネ（10歳）。未婚のまま出産。周囲の反対を押し切って育てた |
| 娘の父（公式） | 「猛烈な恋愛をしたが逃げられた」という説明。実際には存在しない虚構 |
| 娘の父（真実） | 二十四騎討伐戦において将級禍者による性被害の結果懐妊。真実は公表できない |
| 過去の傷（真実） | 18歳の時、将級禍者との戦いで性被害を受けた。生還者全員が察しているが誰も口にしない |
| 公式の嘘の理由 | 真実を語れば将級禍者の存在・撤退組の存在・公式記録の歪曲が全て露見する |
| 「世俗を見ているが深入りしない」姿勢の根底 | 恋愛の傷ではなく、あの夜の真実を抱えて生きてきた10年間 |
| モルダへの感情 | 少々面白くないと感じているが、正面から争う気はない |
| 娘への期待 | ガレスとの縁を内心で望んでいる。ただし動かない |
| 癒着への認識 | 知っている。静観している |

**外見**

| 項目 | 内容 |
|------|------|
| 身長 | やや高め。女性の中では頭一つ抜ける |
| 体型 | 均整の取れた引き締まった体。凛と立つ像のような体型。戦闘できる体だが武人型ではなく女性的な曲線がある |
| 髪色 | プラチナに近い明るい金髪。太陽光の下では白みがかって輝く。「太陽の力の道」という名の体現 |
| 髪型 | 長髪。普段は清潔に編み上げている。儀式・祈りの場では流す |
| 目 | 澄んだ青。感情が揺れても目の静けさが変わらない。「見透かされている」と相手が感じる種類の目 |
| 肌 | 健康的な白。光の中に立つと後光のように輝いて見える |
| 顔立ち | 端正で凛とした美しさ。「近寄りがたい」ではなく「動じない」美しさ。滅多に笑わないが笑った瞬間だけ別人のように柔らかくなる |
| 衣装 | 白と金の半透明ローブ。質素だが仕立てが良い。透け感が神聖さと色香を同時に体現する |
| 雰囲気 | 話しかけにくい。しかし拒絶しているわけではない。太陽のように「ただそこにある」静けさ |
| 色香の質 | 意図していない。神聖さそのものが色香として滲み出る種類。性的な視線を向けた側が勝手に引いてしまう |
| 総合印象 | 美人というより「尊い」。近くにいると場の空気が変わる |

**核心：18歳で経験した真実を誰にも言えないまま、「動かないこと」を選んで生きてきた女性。その静けさは諦めでも恋愛の傷でもなく、10年間抱えてきた重さそのものである。**

## ソルネ（ソルヴェイの娘）

| 項目 | 内容 |
|------|------|
| 名前 | **ソルネ**（語源：古北欧語 *sol*＋*nē*「新たな太陽」） |
| 年齢 | 10歳（メインストーリー時） |
| 母 | ソルヴェイ |
| 父（真実・秘匿） | ラズガルド（混沌と自由の守護神の子・半神）。本人も含め誰も知らない |
| 血の構成 | 母：共鳴能力の高い人間女性。父：守護神の血を引く半神。守護神の血を四分の一引く |
| 能力 | すでに聖女級の奇跡を起こし周囲を驚かせている。母ソルヴェイを凌ぐかもしれないと言われ始めている |
| 聖女認定 | 既定路線だが年齢・分別がつくまで保留 |
| 性格 | 自由奔放・好奇心旺盛（父ラズガルドの影響）。しかし傍若無人さは受け継がず、思いやり深い |
| 物語上の役割 | 淫魔による汚染をまだ少女だったがゆえに免れた次世代の清らかなエース。母が失いかけた「太陽の力」を新たに継ぐ存在 |
| 才能の謎 | 「なぜこれほどの才能が」という疑問は周囲に残る。父が半神であることは最後まで秘密 |

**外見（20歳時のイメージ）**

| 項目 | 内容 |
|------|------|
| 身長 | 母譲りのやや高め。均整の取れた体型 |
| 体型 | 母と似た均整の良い体型。ただし母より少し柔らかみがある |
| 髪色 | 母より少し濃いめの蜂蜜色ゴールド。温かみのある深みのある金 |
| 髪型 | 長髪。母と異なり自然に流すことが多い。自由奔放な性格の反映 |
| 目 | 母譲りの澄んだ青。ただし母の「静けさ」とは違い好奇心と温かみが前に出る |
| 顔立ち | 母の造形を受け継ぎながら、丸みのある若々しさと可愛らしさが加わる。静止していると「美人」、動くと「かわいい」に変わる |
| 衣装 | 白系の聖職者ローブ。母より少し軽やか・動きやすい印象 |
| 母との違い | 同じ神聖な美しさを持ちながら、母の「動じない静けさ」より「開かれた温かさ」が前面に出る。近づきやすい |
| 総合印象 | 「ソルヴェイ様の娘さんだ」と一目でわかる。しかし雰囲気は全く異なる——母が太陽なら娘は春の陽光 |

## 聖モルダ（現No2・元No1）

| 項目 | 内容 |
|------|------|
| 名前 | **モルダ**（語源：ケルト語 *mor*＋*da*「偉大な善」） |
| 称号 | 「聖」称号は保持。ただしNo1の座は4年前にソルヴェイに譲った |
| 年齢 | 45歳 |
| 奇跡の力 | 強力。長年No1を務めただけの実力は今も健在 |
| 教会内の影響力 | 絶大。最高司祭ですら逆らいがたい。長年の蓄積による人脈・権威 |
| 人物像 | 老獪な長老。若い世代を動かすことで存在感を示す |
| No1喪失への感情 | 理性で包んでいるが根に持っている。その感情が行動の底流 |
| イゾルダとの関係 | 執行部を経由して接近。ガレスの婚約候補リリエンを推薦した当事者 |
| 配下との関係 | エヴレン・リリエンを後見。自分の影響下の聖女を世俗に送り込み間接的影響力を維持しようとしている |
| 癒着への認識 | 自らその構造に入った当事者 |

**核心：「偉大な善」という名のまま、善が腐っていく。No1の座は失ったが盤面ごと動かせば結果は同じという発想。しかし老獪さと執念が、純粋な信仰者としての自分をいつの間にか侵食している。**

## 聖エヴレン（No3・モルダ派）

| 項目 | 内容 |
|------|------|
| 名前 | **エヴレン**（語源：ゲルマン系 *ever*＋*ren*「永遠に清らかな静けさ」） |
| 称号 | 「聖」称号持ち（No3相当） |
| 年齢 | 27歳 |
| 婚姻 | 未婚 |
| 所属 | モルダ派。モルダに引っ張られることで動ける |
| 奇跡の力 | 実力はあるが引っ込み思案のため能力を十分に発揮していない |
| 外見 | 非常に美しく男性からも人気がある |
| 男性不信 | 幼少期に男性から身体的な被害を受けたトラウマ。これが引っ込み思案の根底にある |
| ソルヴェイへの感情 | 凛々しい美しさと実力を持つソルヴェイに密かに憧れている。自分がなれなかった姿への羨望 |
| 将来の運命 | 淫魔によって性に目覚め、淫魔の教え（実態はフリダナの教えと大部分が重なる）を後進に普及させる中心人物となる |
| 普及活動の性質 | 「愛と和平の教え」として違和感なく広がる。本人に悪意はなく、むしろ解放感と使命感を持って活動する |
| 物語的な皮肉 | 「永遠に清らか」という名を持つ女性が、最も広く淫魔の教えを広める。悪意のない伝道者であるがゆえに最も深く浸透する |

**核心：幼少期の傷から男性を恐れ閉じたまま「永遠に清らか」でいようとした女性が、淫魔によって初めて「開かれ」、その解放感を善意で後進に広めていく。レイヴァンが救う時、悪意のない伝道者をどう扱うかという問いが生まれる。**

## リリエン（ガレスの婚約候補・モルダ派）

| 項目 | 内容 |
|------|------|
| 名前 | **リリエン**（語源：ラテン *lilium*＋古英語 *rien*「清らかな百合」） |
| 称号 | 「聖」はまだない。「いずれつく」と噂される国内最高クラスの聖女 |
| 年齢 | 17歳 |
| 後見 | モルダが師匠・後見人格 |
| ガレスへの気持ち | 政治的背景は薄々感じているが、**自分の意志でガレスへの婚約を望んでいる** |
| 純粋さの質 | 利用されているという感覚はどこかにある。しかし知りたくないという側面もある |
| 物語的な共鳴 | レイヴァンの「本物の感情か操作か」という主題と響き合う。「自分の気持ちは本物だ」という信念が試される |

**核心：「清らかな百合」という名を持つ少女が、純潔を汚される運命にある。その信念の純粋さが、堕ちる時の落差を最大にする。**
