| 後悔 | 表向きはしていない。しかし無意識に何かを恐れている |

### 対魔神政策体系

神の子計画は「切り札」であり、それ以外にも複数の手を打っている。

| 政策 | 内容 |
|------|------|
| 対禍者特殊部隊の創設 | アルドレスを初代隊長に据えた人事も国王の判断 |
| 魔神出現の早期警戒網 | 国内の魔素濃度の監視・情報網の整備 |
| 騎士団の対魔神訓練強化 | 対人戦闘とは異なる訓練体系の整備 |
| 他国との対魔神協定 | 大規模侵攻に備えた同盟・情報共有 |
| 神官組織との連携強化 | 魔神に対する神聖魔法・封印技術の確保 |
| 神の子計画（第1弾） | 愛妾セリナを神に捧げ、伝説級の英雄を生み出す試み |
| 神の子計画（第2弾） | 今度こそ自身が女神との間に子を作ることを望む。神殿と調整中だが、イゾルダの工作により膠着している |

### 神の子計画・第1弾の経緯（確定版）

**計画の実態：**

アルヴィンは当初「自ら女神と交わって神の子を産みたい（父になりたい）」と望んだが叶わなかった。多くの女神は人間との子を産むことに興味がなく（嫌悪感を持つ者も多い）、神殿の求めに応じなかった。代わりに愛妾セリナを差し出す形で男神への降臨を願ったところ、戦いの守護神**ガルダン**一柱が応じた。

| 時点 | アルヴィンの行動 |
|------|----------------|
| 神託受領（約29歳） | 禍者大侵攻の予言を受け、通常の手段では足りないと判断 |
| 計画立案 | 神殿と秘密裏に交渉。当初は自ら女神と交わることを望んだが実現不可 |
| 方針転換 | 愛妾セリナを差し出す形で男神への降臨を願う |
| ガルダンの応答 | ガルダン（戦いの守護神）一柱が応じた。理由：アルヴィンの「王国を守るために戦う」という意思がガルダンの基準を満たした |
| 神の降臨期間（約1ヶ月） | セリナをガルダンの前に招き入れる。セリナはレイヴァンを受胎（計画は技術的に成功） |
| 淫魔の介入 | 淫魔の上位個体が人間男性に化けてセリナに接近。魔核を植えつける |
| セリナの変容 | 妊娠中に魔核の影響で性欲が制御不能に。本人の意思に反して性交渉を継続 |
| 結果 | セリナ懐妊・レイヴァン誕生。しかし淫魔の介入により計画の成否を判断できなくなった |
| 事後 | 計画の成否を判断できないまま現在に至る。「失敗したかもしれない」という疑念を抱え続けている |
| イゾルダの行動 | 同じガルダンの降臨期間中に、アルヴィンに内緒で別ルート経由でイゾルダ自身もガルダンと交わった。アルヴィンはこれを知らない |

### レイヴァンへの感情

単純な嫌悪ではない。

| 層 | 内容 |
|----|------|
| 神の子ならば | この国を救う英雄になるはずだった存在 |
| 淫魔の子ならば | 自分の計画を踏みにじった敵の落とし子 |
| どちらかわからない | だから直視できない |

レイヴァンを見るたびに「計画の結果を知りたい自分」と「知るのが怖い自分」が交錯する。冷遇は、その葛藤を処理できない男の不器用な逃避でもある。

| 相手 | 関係の質 |

| カルディア | 自分の娘だと思っている。実際は神とイゾルダの子だが、それを知らない |
| ルシアン | 現時点では関心が薄い。芸術を愛する気質を「王には向かない」と見ている |

### 「不貞の子」の噂

| 項目 | 内容 |
|------|------|
| アルヴィンの動機 | 神の子計画の存在を隠す煙幕。「不貞の子」であれば愛妾を神に捧げた事実を隠蔽できる |
| 流布の方法 | 意図的に流し始め、尾鰭背鰭がついて自然発生的に広がった |

### 物語上の機能

**第一幕：** 遠くからレイヴァンの戦果を注視。「試している」段階。

**第二幕：** レイヴァンが権力中枢に近づいた時、初めて正面から向き合わざるを得なくなる。神の子計画第2弾の膠着が続く中で疑念が再燃する。ミレア騒動により判断力が侵食される。

**第三幕：** 全ての真実が明かされた時——

> 「お前は最初から、英雄だった」
> それを言えるか、言えないか。言えたとして、それは謝罪か。告白か。それとも王としての宣言か。

> 不貞の子と噂を流し孤立させた庶子に、今度は娘を差し出す。

> 女神との子を産ませまいと神殿に圧力をかけ続けた。
> しかしすでに神の子は存在していた。
> 自分が孤立させ、不貞の子と噂を流し続けた、あの庶子として。

### 出生の秘密

#### 人間側の計画（神の子計画）確定版

- 神託により禍者の大侵攻が予言された
- 伝説級の英雄は神・精霊との混血でなければ生まれないとされる
- 国王アルヴィンは当初自ら女神と交わることを望んだが、多くの女神が人間との子を産むことを拒んだため叶わなかった
- 代わりに愛妾セリナを差し出す形で男神への降臨を願い、**ガルダン（戦いの守護神）**一柱が応じた
- 計画は極秘。国王と少数の関与者のみが知る

冷遇された庶子が15歳で元服し、死傷率の高い部隊に放り込まれた——その事実を知った時、アルドレスは純粋な義憤を覚えた。同時に、レイヴァンの中に尋常でない素質の気配を武人の直感で感じ取った。

レイヴァンの女性遍歴を知った時、アルドレスは珍しく激怒した。「女性の気持ちを大事にする」という教えが、よりによってその形で実を結んだことへの不器用な後悔も混じっていた。指導内容のどこが間違っていたか説明できないまま、深いため息をつくしかない構造である。以後、定期的にレイヴァンの動向を把握しようとしているが、当のレイヴァンは上手く立ち回っている。

#### 将軍就任後

政治的に排除された形での昇進だったが、アルドレス自身は表向き泰然と受け入れた。レイヴァンから離れることへの懸念を持ちながら——それを顔に出すことは、武人の矜持が許さなかった。遠くから、しかし常に目を向けている。

#### エルヴィナとの結婚

「未亡人を作れない」という信条を自ら破る形になる。その決断はすなわち——**この人のそばで死ぬより、この人のそばで生きることを選んだ**ということである。言葉にはしない。しないが、エルヴィナにはおそらく伝わる。

#### 最終的な関係性（レイヴァンとの）

レイヴァンが王となった時、アルドレスは右腕将軍として国に立つ。師弟でも、親子でもない。しかしそのどちらの要素も含んだ、他に言葉のない関係。

レイヴァンが「閣下」と呼び続けることを、アルドレスは一度だけ「もうその呼び方はよい」と言ったことがある。レイヴァンは「では何と呼べばいいのですか」と返し、二人ともその後黙った。それ以来、ずっと「閣下」のままである。

#### 「ゾクリとする」の正体

リリスが一目でゾクリとさせる理由は、単純な色気ではない。**「この存在は自分をどうにでもできる」という確信が、相手に無意識に伝わるから**である。脅しているわけでも威圧しているわけでもない。ただその存在が「自分はそういう力を持っている」という事実を当然のこととして纏っている。それが表の甘さと組み合わさることで、引き寄せられながらゾクリとするという特有の感覚を生む。

#### 性格

##### 表の顔——「甘い夜の空気」

| 要素 | 具体的な表れ方 |
|------|--------------|
| 声 | 低めで滑らか。静かな場所でだけ聞こえる音楽のような声 |
| 距離感 | 自然に近づく。相手が気づいた時にはすでに近い |
| 笑い方 | 口角だけがわずかに上がる。声を出して笑うことはほぼない |
| 言葉の選び方 | 多くを語らない。短い言葉で相手の思考を占領する |
| 相手への態度 | 誰に対しても同じ温度。特別扱いしているように見えて全員に同じことをしている |
| 沈黙の使い方 | 言葉より沈黙の方が雄弁。間を恐れない |

**重要：** リリスは「誘っている」という素振りを見せない。ただそこにいるだけで相手が勝手に引き寄せられる。だから「嫌味なくらいセクシー」でありながら「溶け込んでいる」という矛盾が成立する。

| 要素 | 具体的な表れ方 |
|------|--------------|
| 判断基準 | 有用か否か。感情は判断に混入しない（レイヴァンへの感情が生まれるまでは） |
| 失敗への対処 | 激昂しない。静かに、完全に、切り捨てる |
| 敵への態度 | 憎まない。軽蔑もしない。ただ排除する |
| 自己評価 | 極めて冷静。自分の限界と能力を正確に把握している |
| 他の淫魔への態度 | 頂点として君臨するが支配を誇示しない。必要な時に示せば十分と知っている |

**表と裏の境界線：** リリスの最も怖い点は、表と裏が完全に分離していないことである。甘い言葉を発しながら冷酷な判断をしている。笑いながら相手を切り捨てる準備をしている。そのどちらが「本当のリリス」かという問いに答えがない。

## 聖女制度

### 基本定義

教会の階層とは**完全に別軸**の称号。守護神への共鳴能力が突出した女性に与えられる。全守護神への奉仕を旨とし、特定の神への帰属ではない。**禁欲の誓いは求められない**。基本的に教会に所属するが、独自の判断・指揮命令権を持つ。

### 認定条件（三つ全て必要）

| 要素 | 内容 |
|------|------|
| ① 奇跡の力 | 守護神への共鳴能力が高く、実際に奇跡を発現している（核心条件） |
| ② 教義・知識 | 神学・儀式・歴史の習熟 |
| ③ 人格・徳行 | 信徒・地域からの信望、日常の行い |

**「神の承認」について：** 守護神は基本的に無関心であり積極的に聖女を任命することはない。ただしまれに特定の人物と守護神が深く共鳴しているという客観的事実が現れることがある。これが「神の承認」として解釈される。神の示しがある場合は教会の認定が事実上の追認となる。

### 称号の種別と権限

| 称号 | 条件 | 権限 | 人数（現在） |
|------|------|------|------------|
| **聖女** | 三要素の認定 | 教会内で自動的に**司祭以上**の扱い。独自の判断・指揮命令権を持つ | 少数（全体から見れば僅か） |
| **聖〇〇**（名前の前に「聖」） | 聖女の中でも特に卓越した共鳴能力・実績 | **高司祭・最高司祭級の決裁権**も持つ | 現在2〜5人程度 |

### 聖女と淫魔将の力関係

| 対象 | 聖女の対抗可否 |
|------|-------------|
| 一般の淫魔 | 抵抗可能 |
| **淫魔将** | **抵抗不可能。人間の限界を超えた存在** |

淫魔将は守護神から権能を預かっており、いかに強力な聖女であっても人間である以上は抗えない。「聖」称号者でさえ例外ではない。**最も神に近い女性たちが、最も組織的に堕とされていく**という構造は、物語のテーマ「形式と実態の逆転」の最大級の体現である。
